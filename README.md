### How to launch
1. **Prerequest**
    You must install node.js, npm and gatsby.js.

2. **Install Dependencies**
    Run commands below.
    ```sh
    cd shunpoco-dev
    npm i
    ```

3. **Run locally**
    Run a command below.
    ```sh
    npm run develop
    ```
    Then, visit `localhost:8000` on your browser.
